Algoritmo x Lógica

Algoritmo é um conjunto de instruções, como uma receita,ou seja, os comandos em um passo a passo. Para que os dados sejam processados, há a necessidade de se utilizar uma linguagem de programação, pois os computadores não executam diretamente os algoritmos.

Lógica de programação é a estruturação de conceitos e regras que orientam como os algoritmos são organizados para solucionar problemas e atingir um resultado desejado.
